Set community settings

• historylimit [1-50]
• welcomemessage [message]

• lang [LANGUAGE_CODE]

• songlimit [SECONDS]

swap
Swap two users in waitlist. Use swap 1 2 or swap @user1 @user2 or combine. swap 8 5
move
Move DJ in waitlist move @user 1
dc
Move you to position where you disconnected. DC command has been updated for better functionality dc
autodc
When someone join to community, Origem Bot automatically check, whether user has been disconnected from Waitlist. (default OFF) autodc on/off
hidecommands
Hide commands. If you write command, Origem Bot delete it and 1 second later write result. hidecommands on/off
skiphistory
Skip song if is in history? skiphistory on/off
roomstats
Show community stats from Origem Bot starts roomstats
download
Download current video (mp3). Soundcloud not supported! download
commandlog
If you use the "hidecommands" command, Origem Bot will record commands. commandlog on/off

Lock DJ booth lock

Unlock DJ booth unlock

Ban current song songban
