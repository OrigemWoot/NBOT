Definition

			
		
				onMention: Origem Bot reply from this when you @mention to bot (only first time, read more below*)

				offMention: Origem Bot reply from this everytime. It's recommeded add here only messages like greeting (hi, hello, bye, cya, brb, afk ..)

				global: Origem Bot reply from this when Origem Bot can't find match message in onMention and offMention and BOT is mentioned.



				msgType is type of message. Available options is only "contains" and "mustBe".

				msgContains is Array with strings. If Origem Bot find one from this strings, Origem Bot automatically reply random string from botReply array.

				botReply is Array with strings what Origem Bot reply.



				msgType: "contains" - Origem Bot reply random message from botReply, where message contains whatever from msgContains array.

				msgType: "mustBe" - Origem Bot reply random message from botReply, where message is 100% match one from msgContains array.

				

			*Mention isn't required only when you are last person who talking with BOT. Only first mention is required. When somebody else post message in chat, you must @mention Origem Bot again.
